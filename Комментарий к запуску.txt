Открыть 3 командных строки:

win+r
cmd
cd пака с файлами client и server

и так 3 раза

1 командная строка:
python server.py

2 командная строка:
python client.py

3 командная строка:
python client.py

Далее мы можем играть
Как играть указано в messagebox-ах,
где выводятся правила игры и
информация об управлении.

В папке есть два варианта: client_comment,server_comment
и client,server.
Первый вариант с комментариями, а второй без.

Хорошой сдачи!
